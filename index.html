ToolBox

ToolBox is single page web app that manages at home DIY projects. Users can manage their budgets, tasks, and completion. The front end was built using ReactJS, and the client communicates with a RESTful API created with ExpressJS, using Mongoose to build schemas in MongoDB. Future iteration will include a feature to place projects in phases and create shopping lists.

Technologies Used: React.js, React Bootstrap, React Router, Express.js, Mongoose, MongoDB, CSS, HTML, AXIOS, Heroku

Swanky Spirits

Swanky Spirits is a single page web app that manages package store inventory, built in collaboration with 3 Developers. Employees can create, update, and delete inventory items and watch the dynamic table update instantly! Future iteration will integrate an API to manage inventory via barcodes.

Technologies Used: React.js, React Bootstrap, React Router, Express.js, Mongoose, MongoDB, CSS, HTML, AXIOS, Heroku

Quick Sale

Quick Sale is a For Sale by Owner house listing web app that allows users to see listings, post listings, and edit and delete their listings. Integrated Google Places API, Address Autocomplete, feature to authenticate addresses. Future iteration will include integration with Amazon S3 to host images.

Technologies Used: Handlebars, Google Places, Express.js, Mongoose, MongoDB, CSS, HTML, AXIOS, Heroku
